# これは最初に読み込まれるストーリーファイルです

#
# メニュー表示
#

# メニューの背景を表示します
@bg file=title.png t=1.0

# メニューを再表示するとき、ここに飛んできます
:MENU

# メニューを表示します
@menu file=title.txt

#
# メニューで"スタート"が選択されたとき、ここにジャンプしてきます
#

:START

# シーン1にジャンプします
@goto label="シーン1"

#
# メニューで"ロード"が選択されたとき、ここにジャンプしてきます
#
:LOAD

# ロード画面を表示します
@menu file=load.txt

# ロードされなかった場合はここに来ます
@goto label="MENU"

#
# メニューで"設定"が選択されたとき、ここにジャンプしてきます
#

:CONFIG

# 設定画面を表示します
@menu file=config.txt

# 設定画面のあとここに戻ってきます
@goto label="MENU"

#
# シーン1 (セリフとモノローグの交錯)
#

:シーン1

@bg coast.png t=0.5

\===
これはノベル作品のサンプルです。
\---
ストーリーファイルでの改行には行頭で --- を使います。
\---
同様に、改ページは行頭で === を使います。
\---

\===
キャラクタを入場させます。
\---

@ch pos=center file=midori/fun.png t=0 effect=normal right=600 down=0 alpha=0
@move pos=center t=2.0 accel=move x=-600 y=0 alpha=255

\===
選択肢は @ichoose を使うとインライン表示を行うことができます。
\---
@ichoose YES はい NO いいえ

:YES
:NO

\===
あとはアドベンチャー形式と同じです。
\---

# メッセージボックスの消去を手動で行います。
@plugin hide-msgbox.txt

@story init.txt
